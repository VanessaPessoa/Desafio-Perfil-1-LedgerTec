<h2> Requisitos </h2>
<p> Ter o node na máquina </p>

<h2> Como executar </h2>

<p> Entrar na pasta back </p>
<p> <b> cd back </b> </p>

<p> Baixar as dependências </p>
<p><b> npm install </b> </p>

<p> Execute o servidor </p>
<p><b> npm start </b> </p>
<p> Agora com o servidor já rodando, entrar na pasta client </p>
<p><b> cd client </b></p>

<p> Baixar as dependências </p>
<p><b> npm install </b></p>

<p> Executar o front </p>
<p><b> npm start </b></p>
